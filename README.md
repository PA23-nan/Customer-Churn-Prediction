# Customer Churn Prediction

This project uses machine learning to predict telecom customer churn.

## Dataset
The dataset contains **7,043 telecom customers** with information on:
- demographics
- contract type
- service usage
- billing details

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Process
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Logistic regression model training
5. Model evaluation

## Results
The logistic regression model achieved approximately **85% accuracy** predicting customer churn.

Key factors influencing churn include:
- contract type
- monthly charges
- customer tenure
